## 📌 Project Overview
This project is a web-based form-filling application that bridges a custom frontend UI with Google Forms and an excel sheet. It allows users to submit data through a designed HTML/CSS interface, which is then automatically stored in a spreadsheet for easy data management.

## 🚀 Features
* **Custom User Interface:** Responsive and clean form design using HTML and CSS.
* **Client-Side Validation:** JavaScript ensures that users enter the correct data before submitting.
* **Automated Data Sync:** Form submissions are linked directly to Google Forms/Excel.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Version Control:** Git & GitHub
